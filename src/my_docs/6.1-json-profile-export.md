# Пункт 6.1: экспорт профиля выполнения в JSON

## Как реализовано
Профиль собирается как `Dictionary<string, object?>` и помещается в `OperationResult.Extensions` под ключом `profiling`. Дальше он сериализуется в JSON стандартным GraphQL-ответом.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/src/Types/Execution/Pipeline/ExecutionProfilerMiddleware.cs`

## Ключевые цитаты
```csharp
public const string ExtensionKey = "profiling";
```

```csharp
var result = new Dictionary<string, object?>
{
    ["requestDurationNs"] = snapshot.RequestDurationNanoseconds
};
```

```csharp
operationResult.Extensions =
    operationResult.Extensions.SetItem(
        ExecutionProfileCollector.ExtensionKey,
```
