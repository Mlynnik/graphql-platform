# Пункт 4.5: динамическое включение/отключение профилирования

## Как реализовано
Поддержаны два уровня управления:
1. Runtime-state через singleton `IExecutionProfilerState`.
2. Request-level override через `ExecutionProfilerRequestOverrides` и `OperationRequestBuilder`.

## Ключевые цитаты
```csharp
public bool IsEnabled => Volatile.Read(ref _isEnabled) == 1;

public void SetEnabled(bool enabled)
{
    Interlocked.Exchange(ref _isEnabled, enabled ? 1 : 0);
}
```

```csharp
public static OperationRequestBuilder DisableExecutionProfiler(
    this OperationRequestBuilder builder)
    => builder.SetExecutionProfilerEnabled(enabled: false);
```

```csharp
if (context.Features.TryGet<ExecutionProfilerRequestOverrides>(out var requestOverrides)
    && requestOverrides.IsEnabled is { } requestScopedState)
{
    return requestScopedState;
}
```
