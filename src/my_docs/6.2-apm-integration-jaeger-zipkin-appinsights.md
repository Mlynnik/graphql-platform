# Пункт 6.2: интеграция с внешними APM (Jaeger/Zipkin/Application Insights)

## Как реализовано
Интеграция построена через OpenTelemetry API:
- `Meter` для метрик,
- `ActivitySource` для трассировок.

Дальнейшая отправка в Jaeger/Zipkin/Application Insights выполняется через стандартные OTel exporters в окружении приложения.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerOpenTelemetryExporter.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerTelemetry.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`
