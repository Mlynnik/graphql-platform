# Пункт 2.4: рекомендации по использованию DataLoader

## Как реализовано
Для каждой найденной потенциальной N+1-проблемы формируется прямой текст рекомендации в issue-пayload.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
["message"] =
    "Potential N+1 pattern detected for repeated list field resolution without DataLoader batching.",
["recommendation"] = "Use DataLoader to batch this field access by parent keys."
```

```csharp
if (patternMetric.DataLoaderBatchCalls > 0
    || patternMetric.DataLoaderCacheHits > 0)
{
    continue;
}
```

Эта проверка предотвращает рекомендацию там, где batching/cache уже реально используются.
