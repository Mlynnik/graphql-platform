# Пункт 4.3: интеграция с OpenTelemetry для экспорта метрик

## Как реализовано
Экспортер `ExecutionProfilerOpenTelemetryExporter` публикует:
- метрики через `Meter` (`Counter`, `Histogram`),
- трассировки через `ActivitySource`.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerOpenTelemetryExporter.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerTelemetry.cs`
