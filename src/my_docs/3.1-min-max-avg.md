# Пункт 3.1: min/max/avg времени выполнения

## Как реализовано
В агрегаторе есть `DurationAccumulator`, который накапливает длительности и вычисляет `minNs`, `maxNs`, `avgNs`.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerSlidingWindowAggregator.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
private long _min = long.MaxValue;
private long _max = long.MinValue;
private double _sum;
```

```csharp
public void Add(long durationNanoseconds)
{
    _samples.Add(durationNanoseconds);
    _sum += durationNanoseconds;
    if (durationNanoseconds < _min) { _min = durationNanoseconds; }
    if (durationNanoseconds > _max) { _max = durationNanoseconds; }
}
```

```csharp
["minNs"] = _min,
["maxNs"] = _max,
["avgNs"] = _sum / _samples.Count,
```
