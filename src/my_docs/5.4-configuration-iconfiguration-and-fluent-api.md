# Пункт 5.4: конфигурация через `IConfiguration` и fluent API

## Как реализовано
Поддержаны оба сценария:
- `AddExecutionProfiler(configuration, sectionPath)` через `IConfiguration`.
- `AddExecutionProfiler(options => ...)` и `ModifyExecutionProfilerOptions(...)` через fluent API.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/DependencyInjection/RequestExecutorBuilderExtensions.Profiling.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerServiceCollectionExtensions.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
public static IRequestExecutorBuilder AddExecutionProfiler(
    this IRequestExecutorBuilder builder,
    IConfiguration configuration,
    string sectionPath = ExecutionProfilerOptions.DefaultConfigurationSectionPath)
```

```csharp
builder.Services.ConfigureExecutionProfiler(configuration, sectionPath);
return builder.AddExecutionProfiler();
```

```csharp
public static IRequestExecutorBuilder ModifyExecutionProfilerOptions(
    this IRequestExecutorBuilder builder,
    Action<ExecutionProfilerOptions> configure)
{
    builder.Services.ConfigureExecutionProfiler(configure);
    return builder;
}
```
