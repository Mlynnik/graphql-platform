# Пункт 1.4: сбор статистики кеша DataLoader (hits/misses)

## Как реализовано
- `cache hit` фиксируется по событию `ResolvedTaskFromCache`.
- `cache miss` считается через размер `keys` в batch-вызове (`AddDataLoaderBatch`).
- Метрики собираются как на request-level, так и на field-level.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerDataLoaderDiagnosticEventListener.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
public override void ResolvedTaskFromCache(
    IDataLoader dataLoader,
    PromiseCacheKey cacheKey,
    Task task)
{
    if (TryGetCollector(out var collector))
    {
        collector.AddDataLoaderCacheHit(ExecutionProfilerScopeContext.CurrentPath);
    }
}
```
