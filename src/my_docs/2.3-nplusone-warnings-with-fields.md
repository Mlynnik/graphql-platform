# Пункт 2.3: предупреждения с указанием полей, вызывающих проблему

## Как реализовано
При срабатывании детекции формируется payload `nPlusOne` в profiling extension: общее число проблем и список issues с указанием проблемного `pathPattern` и `examplePath`.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
result["nPlusOne"] = new Dictionary<string, object?>
{
    ["issueCount"] = nPlusOneIssues.Length,
    ["issues"] = nPlusOneIssues
};
```

```csharp
issues.Add(new Dictionary<string, object?>
{
    ["pathPattern"] = entry.Key,
    ["occurrences"] = patternMetric.Occurrences,
    ["examplePath"] = patternMetric.ExamplePath,
    ["message"] = "Potential N+1 pattern detected for repeated list field resolution without DataLoader batching.",
    ["recommendation"] = "Use DataLoader to batch this field access by parent keys."
});
```
