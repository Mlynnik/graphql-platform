# Пункт 5.2: исключение определенных типов/полей из профилирования

## Как реализовано
В `ExecutionProfilerOptions` есть наборы исключений:
- `ExcludedObjectTypes`
- `ExcludedFieldCoordinates`
- `ExcludedFieldNames`

Проверка выполняется в `ShouldProfileField(...)` до записи field-метрики.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerOptions.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
public ISet<string> ExcludedObjectTypes { get; } = new HashSet<string>(StringComparer.Ordinal);
public ISet<string> ExcludedFieldCoordinates { get; } = new HashSet<string>(StringComparer.Ordinal);
public ISet<string> ExcludedFieldNames { get; } = new HashSet<string>(StringComparer.Ordinal);
```

```csharp
if (ExcludedObjectTypes.Contains(objectType)
    || ExcludedFieldCoordinates.Contains(coordinate)
    || ExcludedFieldNames.Contains(fieldName))
{
    return false;
}
```

```csharp
if (_options?.ShouldProfileField(pathValue, coordinate, objectType, fieldName) == false)
{
    return;
}
```
