# Пункт 7.4: тестирование при высокой нагрузке (concurrent requests)

## Как реализовано
Добавлен сценарий массового параллельного выполнения запросов с проверкой:
- наличие `profiling` extension в каждом ответе,
- корректного размера окна агрегатора после конкурентной нагрузки.

## Основные файлы
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
const int requestCount = 64;
...
var tasks = new Task<IExecutionResult>[requestCount];
for (var i = 0; i < requestCount; i++)
{
    tasks[i] = executor.ExecuteAsync("{ child { nested { name } } }");
}
```

```csharp
var results = await Task.WhenAll(tasks);
...
Assert.True(HasProfilingExtension(operationResult));
```

```csharp
var statistics = executor.GetExecutionProfilerStatistics();
var window = GetDictionaryValue(statistics, "window");
Assert.Equal(requestCount, GetIntValue(window, "requestCount"));
```
