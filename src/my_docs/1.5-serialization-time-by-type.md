# Пункт 1.5: измерение времени сериализации результата для каждого типа

## Как реализовано
Замер добавлен вокруг `type.CoerceOutputValue(...)` в `ValueCompletion.Leaf`. После сериализации метрика записывается в коллектор по имени GraphQL-типа (`type.Name`).

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Processing/ValueCompletion.Leaf.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
var serializationStartTimestamp = Stopwatch.GetTimestamp();

try
{
    type.CoerceOutputValue(runtimeValue, resultValue);
}
finally
{
    collector.AddSerializationByType(
        resultValue.Path,
        type.Name,
        Stopwatch.GetElapsedTime(serializationStartTimestamp).Ticks * 100);
}
```

```csharp
var metrics = GetOrCreateSerializationMetrics(typeName);
metrics.Count++;
metrics.TotalDurationNanoseconds += durationNanoseconds;
```
