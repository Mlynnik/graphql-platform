# Пункт 1.2: отслеживание глубины дерева выполнения для каждого поля

## Как реализовано
Глубина считается из `Path`: учитываются только `NamePathSegment`, что дает корректный depth по полям GraphQL.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
var field = new ExecutionProfileFieldEntry(
    pathValue,
    GetFieldDepth(path),
    durationNanoseconds,
    coordinate,
    objectType,
    fieldName);
```

```csharp
private static int GetFieldDepth(Path path)
{
    var depth = 0;
    var current = path;

    while (!current.IsRoot)
    {
        if (current is NamePathSegment)
        {
            depth++;
        }

        current = current.Parent;
    }

    return depth;
}
```
