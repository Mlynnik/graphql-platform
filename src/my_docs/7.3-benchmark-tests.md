# Пункт 7.3: benchmark tests профилировщика

## Как реализовано
Добавлен BenchmarkDotNet benchmark для сравнения накладных расходов:
- baseline: `Execute_WithProfilerOff`
- сравнение: `Execute_WithProfilerOn`

- `HotChocolate/Core/benchmarks/Execution.Profiling.Benchmarks/ExecutionProfilerOverheadBenchmark.cs`
- `HotChocolate/Core/benchmarks/Execution.Profiling.Benchmarks/Program.cs`
