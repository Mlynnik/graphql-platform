# Пункт 5.5: фильтрация по пути запроса (profile только определенные queries/paths)

## Как реализовано
Профайлер поддерживает `IncludedPathPrefixes`: если список непустой, профилируются только пути, совпавшие с префиксом. Для list-путей используется нормализация индексов.
