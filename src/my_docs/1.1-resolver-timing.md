# Пункт 1.1: измерение времени выполнения каждого resolver (sync/async)

## Как реализовано
Время поля измеряется в диагностическом listener на событии `ResolveFieldValue`. Подход работает одинаково для sync и async resolver, потому что измерение оборачивает весь жизненный цикл field scope.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerDiagnosticEventListener.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfileCollector.cs`
- `HotChocolate/Core/test/Execution.Tests/DependencyInjection/RequestExecutorBuilderExtensions_ExecutionProfiler.Tests.cs`

## Ключевые цитаты
```csharp
public override IDisposable ResolveFieldValue(IMiddlewareContext context)
{
    ...
    return new ResolveFieldScope(...);
}
```

```csharp
private readonly long _startTimestamp = Stopwatch.GetTimestamp();
...
_collector.AddField(
    _path,
    Stopwatch.GetElapsedTime(_startTimestamp).Ticks * 100,
    _coordinate,

    _fieldName);
```
