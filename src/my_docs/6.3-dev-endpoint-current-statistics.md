# Пункт 6.3: endpoint для получения текущей статистики (только для разработки)

## Как реализовано
Добавлен endpoint `MapGraphQLProfiler` (по умолчанию `/graphql/profiler`):
- в `Development` отдает JSON-статистику агрегатора,
- вне `Development` возвращает `404`.

## Основные файлы
- `HotChocolate/AspNetCore/src/AspNetCore/Extensions/HotChocolateAspNetCoreEndpointRouteBuilderExtensions.ExecutionProfiler.cs`
- `HotChocolate/Core/src/Types/Execution/Extensions/ExecutionProfilerRequestContextExtensions.cs`

## Ключевые цитаты
```csharp
private const string GraphQLProfilerPath = "/graphql/profiler";
```

```csharp
if (!(environment?.IsDevelopment() ?? true))
{
    context.Response.StatusCode = StatusCodes.Status404NotFound;
    return;
}
```

```csharp
var statistics = executor.GetExecutionProfilerStatistics();
await context.Response.WriteAsJsonAsync(statistics, context.RequestAborted);
```
