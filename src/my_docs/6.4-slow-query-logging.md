# Пункт 6.4: логирование медленных запросов с детализацией полей

## Как реализовано
В `ExecutionProfilerMiddleware` реализован `LogSlowRequestIfNeeded`, который:
- проверяет `SlowRequestLoggingEnabled` и порог `SlowRequestThreshold`,
- выбирает top-N медленных полей (`SlowRequestFieldLimit`),
- пишет `LogWarning` с operation info и деталями DataLoader.

## Основные файлы
- `HotChocolate/Core/src/Types/Execution/Pipeline/ExecutionProfilerMiddleware.cs`
- `HotChocolate/Core/src/Types/Execution/Profiling/ExecutionProfilerOptions.cs`
